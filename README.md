# Voice RAG Pipeline - Fully Offline

A complete voice-based Retrieval-Augmented Generation system running entirely offline on Linux, optimized for CPU-only processing with 16GB RAM.

## Project Location

All project files are located in the `Open_Source_RAG_With_Speech/` directory.

Please navigate to that folder for the complete application:

```bash
cd Open_Source_RAG_With_Speech
```

See `Open_Source_RAG_With_Speech/README.md` for full documentation, setup instructions, and usage guide.
